# My CV Template

A simple and customizable **CV template built with HTML and CSS**, designed for easy editing and PDF generation.

---

## Getting Started

This project uses **[Vite](https://vite.dev/)** for fast development and **[Tailwind CSS](https://tailwindcss.com/)** for styling.

### Prerequisites

- Node.js (v16 or later recommended)
- npm

---

## How to Run Locally

1. Install project dependencies:

```
npm install
```

2. Start the Vite development server and open the CV in your browser:

```
npm run dev
```

This will launch a live server where you can edit `index.html` and see changes instantly.

---

## Generate PDF

The project includes **`pdf.js`** for generating a PDF version of the CV.

To generate the PDF:

1. Open a terminal in the project root
2. Run:

```
node pdf
```

A PDF version of the CV will be generated automatically.

---

## Customization

- Update your content in `index.html`
- Modify styles using Tailwind utility classes
- Adjust layout or typography as needed
